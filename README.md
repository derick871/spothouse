# Spothouse
It is a website that has identified the problem of house
seeking and come up with the solution.

## Problem solving
I have identified that people take long time and face
difficulties on seeking houses in urban centers after they have 
been relocated.

## Solutions
I've come up with this project to simplify the process of house searching
by providing accurate location,direct communication with the Landlord
standard of the house and price per month both in one place.

  tailwind

  vs code
